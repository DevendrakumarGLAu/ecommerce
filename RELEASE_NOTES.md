# Release Notes

## 0.0.1

Initial release of the `Devanjalibangles` ecommerce storefront.

Highlights:
- Angular 19-based storefront with server-side rendering (SSR) support
- Product catalog, product details, and responsive shopping experience
- Persistent cart drawer and checkout-ready cart service setup
- Site layout with header, footer, loader, and snackbar notification system
- Build and development scripts configured with Angular CLI and SSR support

Notes:
- Includes `src/app/common`, `src/app/home`, `src/app/product`, and `src/app/product-details`
- Application scaffold supports Angular routing, server entrypoint, and pre-render setup
- Ready for local development via `ng serve` and production build via `ng build`/`ng run` commands
